App Description:

Geek Out

Is originally a card game asking players questions from a broad range of genres. However, our project implements this idea in the android app while giving users answer choices

It features an authentication system which allows multiple-validated users to access the game with their own profiles, including points they earned

Once a user registers, the user's information will be stored in the firebase database

There are two modes: Single Play and Multi Play

Single Play is very similar to Multi Play, but allows user to gain points while practicing

Multi Play mode is in real-time session, letting two players compete each other.

The user's points will be assessed after each game has finished and the account profile will be updated

The points will be used for the ranking system
